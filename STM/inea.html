EA机型操作流程（GSD）
PCC操作流程(auto clea)：

第一步：准备工作
1.VSEN命令产看今天有没有 -MSIMPSURF(PCC) ,红色盒子wafer.
2.清空机器中的产品，
3.BGN-QUATASK 
第二部: 参数值设置
1.打开Beam 菜单
    点击Auto Tune 
    点击Hold+About(多按几次) 变成Idle.
2.GAS(气体)——>off(关)
    点击 Select Specles 
    点选GAS...1(第一个 ARGON) 变成白点就选中了 
  GAS——> ON(开)
    sccm ——> set: 1  每次设置完 要按enter键
3.(第二排) 
  Magnet——> set: 5
  Arc  ——> Actual  set: 0.5
       ——>		   set: 60
4.(第三排) 
  Extraction ——>set:50 
  Gap Axls   Rosn= >=1500
5.(第四排) AMU
  set Amu ——>set:40 
  Acceleration  Actual ——> set :50  
  设置完成
6.点击下面 AMU  看提示 : 忘记什么提示了
  Operator  首页第一个  回到首页
7.Detall ——> Show Auto clean.  
  按一次 Auto clean （会循环10次 ） 
  等待十次循环完成。。。。。。
  
第三部：跑DAMMY -单独跑
1.确认接下来的产品 使用的气体。
2.Add cassetle 菜单选择 
  1.pcc-CONDITIONING   ——>BF3(B+)
                       ——>PH3(p+)
  2.菜单上面 输入ID  任意内容
  3.关门
  
第四部：跑PCC - 单独跑
1.确认接下来的产品 使用的气体。
2.Add cassetle 菜单选择 
  1.pcc   ——>(B+) BORONPCC
          ——>(p+) PHPCC
          ——>(A+) ....
		  
  2.菜单上面 输入ID  任意内容
  3.关门 (不要让PCC精品接触多余的空气)
  4.取出送走。
