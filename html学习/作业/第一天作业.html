PHP介绍
php现状及未来发展趋势；
  1. 全球5000 万互联网网站中，有60%以上使用着PHP技术；
  2. 2010年PHP从业人数增加42%，远超JAVA的13%和NET的24%；
  3. PHP是全球五大最受欢迎的编程语言之一，并且是唯一入选的脚步语言；
  4. 国内80%以上的动态网站使用PHP开发；
  5. AlexaTOP500中国网站排名，有394家使用了PHP技术，比例为78.8%。
PHP以其灵活高效等特点实现了各种终端和平台的互联网应用，源于其具
有如下特点；
  1. 基于Web量身定制
  2. LAMP架构安全稳定
  3. 简单快速，扩展性强
  4. 企业级应用发展迅速
在与其他同类编程语言的比较中，PHP具有开发速度块、运行效率高、安全性
好、可扩展性强、开源自由等特点；而回顾PHP近10年来的发展过程、展望未
来互联网行业的发展趋势，我们可以得出结论：PHP的发展势头不可阻挡，比
将成为未来WEB开发领域的主流技术体系。
PHP就业现状

随着IT就业和互联网的速度发展，企业对PHP程序员的需求也大量增加，PHP程
序员和招聘岗位的供求比例是1：40，很多公司半年都招不到一个合适的PHP程
序员。这个岗位是程序员中最火的，这种严重供应不求的局面在未来几年中将
愈演愈烈。

HTML的基本概念
HTML 语言是一种超文本的标记语言,简单来讲就是构建一套
标记符号和语法规则,将所要显示出来的文字、图象、声音等
要素按照一定的标准要求排放，形成一定的标题、段落、列
表等单元。
CSS层叠样式表
HTML 层叠样式表(英文全称：Cascading Style Sheets)是一
种用来表现HTML（标准通用标记语言的一个应用）或XML
（标准通用标记语言的一个子集）等文件样式的计算机语言。
CSS不仅可以静态地修饰网页，还可以配合各种脚本语言动
态地对网页各元素进行格式化。
JavaScript脚本程序
JavaScript一种直译式脚本语言，是一种动态类型、弱类型、基
于原型的语言，内置支持类型。它的解释器被称为JavaScript
引擎，为浏览器的一部分，广泛用于客户端的脚本语言，最早
是在HTML（标准通用标记语言下的一个应用）网页上使用，用
来给HTML网页增加动态功能。
